Static html content is served on http://localhost:8000/restaurant/
Static content can is also served on http://localhost:8000/ with redirect to http://localhost:8000/restaurant/
Menu is served on http://localhost:8000/restaurant/menu/
Menu items is served on http://localhost:8000/restaurant/menu_items/
Single menu items is served on http://localhost:8000/restaurant/menu_items/1
Table booking list is served on http://localhost:8000/restaurant/booking/tables/

User registration is served on http://localhost:8000/auth/users/
